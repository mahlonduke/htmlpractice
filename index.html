This page is being used to practice HTML. This is important because:

  • HTML is the most commonly-used language.
  • While there are WYSIWYG tools innumerable for creating web content without writing raw code, these tools are imperfect. Sometimes you'll need to fix or customize them.
  • Not every project will be so complex that it needs complex tools and hosting. Quickly writing up a new page will be much easier in these cases.
